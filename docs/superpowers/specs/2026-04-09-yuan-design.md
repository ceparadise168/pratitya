# 緣 (Yuán) — 設計文件

> 一個活著的、有情緒的場域，眾生在其中因緣聚散，永不重複。

## 哲學基礎

這是一件純視覺、無聲的生成式藝術作品，以演算法體現佛教核心概念：

- **緣起性空** — 萬物因條件聚合而生，條件散去而滅，無自性
- **無常** — 沒有任何狀態會持續，包括「無常」本身
- **佛相** — 佛有多種面相（忿怒、慈悲、歡喜、禪定、寂靜），每一相都是真實的顯現

作品不是在「描述」這些概念，而是讓觀者直接**體驗**它們。

## 使用場景

- 數位藝術展覽：投影或大螢幕展示，觀眾駐足欣賞
- 個人冥想靜觀：打開後放著，進入沉浸式觀看

無互動。無聲音。觀者唯一的動作是「看」與「不看」。

## 核心架構：場域與眾生 (Field + Agents)

### 場 (Field)

一個不可見的連續場，擁有隱藏參數：

- **能量 (energy)** — 場域的總活性，影響速度與強度
- **溫度 (warmth)** — 色彩的冷暖傾向
- **旋轉 (rotation)** — 旋渦傾向
- **流向 (flow)** — 方向性的力
- **凝聚 (cohesion)** — 聚集或離散的傾向
- **脈動 (pulse)** — 節律性的強弱

這些參數由「相」(aspect) 驅動，並持續緩慢漂移。觀者看不到這些參數，只能看到它們在眾生身上的表現。

### 眾生 (Agents)

一群通用粒子。它們沒有固定的身份——不是「魚」也不是「鳥」。它們的行為完全由場的條件決定：

- 場的凝聚力高＋流向一致 → 看起來像魚群或鳥群
- 場的旋轉力強 → 看起來像漩渦
- 場的脈動高＋能量集中 → 看起來像心跳
- 場的流向平滑 → 看起來像河流
- 場的能量極低 → 看起來像星塵，或幾乎不可見

沒有「場景切換」。只有場的參數連續變化，眾生的行為自然跟隨——這就是**相變 (phase transition)**。

## 佛相：情緒作為場域的維度

場域擁有情緒狀態——「相」。相不是標籤，是參數的組合傾向：

### 忿怒相

- 場域狀態：高能量、高飽和、快速、震盪
- 視覺表現：粒子加速衝撞、強烈飽和色彩、尖銳軌跡、畫面有壓迫感
- 色彩傾向：高飽和暖色（紅、橘、金）

### 慈悲相

- 場域狀態：溫暖、柔軟、緩慢擴散
- 視覺表現：粒子緩慢移動、柔和光暈、彼此靠近但不碰撞、如水般流動
- 色彩傾向：低飽和暖色（淡金、柔橘、暖白）

### 歡喜相

- 場域狀態：豐盛、明亮、輕快
- 視覺表現：大量粒子湧現、色彩相對繽紛、跳動的節奏、向上揚升
- 色彩傾向：中飽和多色（但仍和諧）

### 禪定相

- 場域狀態：極慢、高度有序、中心化
- 視覺表現：幾何般的規律運動、向心收斂、深沉而靜定
- 色彩傾向：低飽和冷色（靛藍、深紫、墨色）

### 寂靜相

- 場域狀態：近乎空無
- 視覺表現：幾乎看不到任何東西，但場域仍在運作——像冬天地底的種子
- 色彩傾向：趨近無色

## 生命週期：呼吸節奏

每一次「生命」遵循呼吸般的節奏：

```
緣起 (Birth) → 存在 (Living) → 緣滅 (Dying) → 寂靜 (Silence) → 再生 (Rebirth)
```

### 一期一會

每次緣起是一場完整的生命。有誕生的湧現、存在的豐盛、衰敗的消散、寂滅的空寂。觀者對每一次緣起產生情感，然後練習放下。

### 時間尺度的無常

持續時間不固定：

- 有的生命如曇花，數秒即滅
- 有的如山河，停留數分鐘
- 觀者永遠無法預測「這次會持續多久」

這本身就是無常的直接體驗。

## 因緣邏輯

### 殘餘條件

場域的相不是隨機切換。每一個相結束時，留下**殘餘條件**影響下一個相的誕生：

- 忿怒相耗盡能量 → 可能轉為寂靜（耗竭），也可能餘怒未消進入另一波忿怒
- 慈悲相可能慢慢積蓄成歡喜，也可能沉入禪定
- 寂靜相中蓄積的條件，決定下一次緣起的性質

每次路徑不同。不是隨機，是**條件的自然流動加上機率**。

### 觀者的心理路徑

```
觀看 → 感受到脈絡 → 試圖預測 → 預測有時對有時錯 → 放棄預測 → 純粹觀看 → 心流
```

這正是冥想的過程。

## 四個藝術層次

### 一、痕跡 (Traces)

每一次生命不完全消失。它留下極淡的視覺殘影——像沙灘上的腳印、焚香後的灰。

- 下一個生命誕生時，前世的殘影隱約可見
- 再下一世，殘影更淡
- 三世之後，完全不可見

這讓因緣不只是看不見的演算法參數，而是**觀者用眼睛可以感受到的記憶層**。新生命在舊生命的幽靈上起舞——這就是業力的視覺化。

實現方式：寂靜相開始時，將當前畫面以極低透明度「烙印」到一個持久的 canvas 層。每一幀對這個持久層施加極緩慢的衰減。

### 二、真正的空 (True Emptiness)

有些寂靜段落中，螢幕是**真正的黑暗**。不是「幾乎空」——是完全的黑。沒有光點、沒有呼吸、沒有任何視覺元素。

- 持續時間不固定：可能 30 秒，可能 2 分鐘
- 不頻繁出現，但會出現
- 觀者會不安、會懷疑、會等待、會放下期待
- **放下期待的那一刻，就是整件作品的核心體驗**

不是每次寂靜都是真空。大部分寂靜仍有微弱的場域活動。真正的空是稀有的——這讓它有重量。

### 三、雨季與旱季 (Macro Rhythm)

單一生命有呼吸節奏。但整個場域也有更大尺度的呼吸：

- **雨季**：生命接連湧現，前一個還未完全消亡，下一個已在其中萌芽。豐盛、繁忙、層層疊疊。
- **旱季**：漫長的寂靜之間只出現短暫而微弱的生命。稀疏、安靜、等待。

觀者無法知道自己正處於雨季還是旱季，只能在回顧時感受到。

這是**劫**的尺度——大時間包裹著小時間。

實現方式：一個緩慢漂移的「豐盛度」參數，影響生命的持續時間、寂靜的長度、以及粒子的數量上限。

### 四、佛性 (Buddha-nature)

一個極其隱微的視覺元素——也許是畫面中某處一個像素大小的微光。

- 它在所有生命中都存在
- 忿怒相時它在，寂靜相時它在
- 真正的空無時，它也在
- 觀者若看得夠久、夠仔細，會發現它
- 它不生不滅、不增不減

這是覺知本身——見證一切生滅的那個「觀」。

然後，非常偶爾——連它也消失。

因為連「永恆的覺知者」這個概念本身，也是空。

## 唐卡視覺語法 (Thangka Visual Grammar)

唐卡不是風格參考，而是一套**視覺語法**——神聖感的結構性來源。以下四個元素抽象化後融入場域的行為邏輯，作為特定條件下的自然顯現，而非裝飾。

### 背光 (Aureole)

唐卡中佛像身後放射狀的光環，讓平面產生維度感——暗示不可見的力量正在向外輻射。

- 觸發條件：場域能量到達臨界點時
- 表現：眾生的軌跡自然地從某個中心向外放射，形成瞬間的對稱放射結構
- 不是繪製光圈，而是粒子行為自發形成的背光
- 出現數秒後破碎消散——你看到了，但無法留住

### 曼陀羅 (Mandala)

同心圓、對稱、層層嵌套——唐卡最核心的幾何。沙曼陀羅更是無常的極致象徵：耗時數週繪製，完成後立刻掃毀。

- 觸發條件：禪定相中，凝聚力與秩序極高時
- 表現：眾生短暫排列成曼陀羅般的對稱幾何圖形——精密、美得令人屏息
- 場的參數漂移後，對稱開始瓦解，曼陀羅溶解回混沌
- 這就是沙曼陀羅的精神——用演算法重現「精心建構，然後放手」

### 火焰紋 (Flame Motif)

忿怒明王身後的火焰——不是裝飾，是能量的具象化。焰的形態有獨特韻律：尖銳、向上、捲曲、反覆分叉。

- 觸發條件：忿怒相中，能量與飽和度極高時
- 表現：粒子軌跡帶有火焰般的捲曲特徵——不是模擬真實火焰，而是唐卡式的、程式化的、帶有裝飾性韻律的焰形曲線
- 軌跡尖銳上揚、分叉、捲繞，具有書法般的力道

### 金 (Gold)

唐卡用金不為華麗，而是標記**超越世俗的維度**。金色意味著「此元素不屬於日常世界」。

- 觸發條件：極其稀有——特定因緣條件完美聚合的罕見時刻
- 表現：粒子色彩短暫偏向金色光芒
- 頻率：觀者可能整整一小時才遇到一次
- 金色本身成為一個**事件**，而非一種配色

## 永不重複

時間戳（精確到毫秒）作為隨機種子的一部分。2026 年 4 月 9 日晚上 8:32:17.483 的這一刻，宇宙中只有這一次。這件作品在那個瞬間的樣子，也只有這一次。

不使用可重播的偽隨機序列。每一幀的決策都混入 `Date.now()` 或 `performance.now()` 的不確定性。

## 技術選擇

- **單一 HTML 檔案**，純 Canvas 2D（或考慮 WebGL 以獲得更好的效能和視覺效果）
- **無外部依賴**——作品本身即完整
- **全螢幕**，無 UI、無控制項、無文字（展覽環境）
- **響應式**——適應任何螢幕比例
- **效能目標**：60fps，支援長時間運行（數小時）不退化

## 不包含

- 聲音（沉默即表達）
- 使用者互動（觀者的唯一動作是看與不看）
- 場景名稱或任何文字覆蓋
- 預設的色彩風格（色彩是相的自然產物）
- 固定的場景序列或敘事弧線
