# 緣 (Yuán) v4 — 數位生成式唐卡

> 以唐卡的構圖法則和宇宙觀為骨架，用演算法生成的視覺語言為血肉，讓觀者直接體驗緣起、無常、空。

---

## 本質定義

不是「用電腦畫唐卡」。不自稱「唐卡」。

這是一件生成式藝術作品，受唐卡傳統的構圖法則和宇宙觀啟發。觀者不需要佛學知識。認識唐卡的人會看出淵源，不認識的人會感受到莊嚴。

---

## 核心結構：同心三層

以極座標系統管理空間。圓形構圖——任何角度觀看都成立。

### 核心層（法度）

度量經比例的抽象幾何格線。

- 來自藏傳唐卡佛像比例骨架的數學關係——交叉線、圓弧、比例分割
- 佛像本身被拿掉，只留下空的格線。「空即是色」
- 幾乎不動。僅有極微弱的呼吸——擴張/收縮週期約 30-60 秒，肉眼需要長時間凝視才能察覺
- 金色細線，低透明度
- 預計算至離屏 canvas，極低頻率更新

### 中層（生滅）

主要的生成式活動區。

- 從核心向外的梯度：越靠近中心越穩定有序，越往外越混沌無常
- 用唐卡的視覺原則生成動態紋樣：
  - **線條品質：** 精細、流暢、有韻律的貝茲弧線。演算法自身的精確美，不模仿手繪
  - **空間密度：** 密而不亂。紋樣充分填滿但有清晰結構。每一寸空間有意圖
- 場域驅動（沿用 v3 的 energy × order 二維連續場域）
- 生成元素不是可辨認的佛教符號——是抽象的幾何紋樣，用弧線和直線構成

### 邊界層（虛空）

- 無硬邊框。紋樣向外漸隱消融入黑暗
- 偶爾有痕跡殘留（trace canvas）
- 邊界是作品與觀者世界的過渡

## 色彩系統

### 唐卡礦物色調色盤

不用即時運算色彩。預定義嚴格的配色規則：

**主色組（高飽和礦物色）：**
- 深藍（群青）：hsl(220, 75%, 25%)
- 朱紅（硃砂）：hsl(5, 80%, 45%)
- 翠綠（石綠）：hsl(155, 65%, 30%)
- 金色（金箔）：hsl(45, 85%, 55%)
- 深紫（紫礦）：hsl(280, 60%, 25%)
- 暖白（白堊）：hsl(40, 30%, 85%)

**配對法則（色彩並置，非漸層）：**
- 金 + 深藍（最經典的唐卡配色）
- 朱紅 + 翠綠
- 暖白 + 深紫
- 金 + 朱紅（稀有——華麗的頂點）

**場域映射：**
- 高能量場域：暖色主導（朱紅、金）
- 低能量場域：冷色主導（深藍、深紫）
- 高秩序：金色線條更明顯
- 低秩序：色彩混濁、邊界模糊

### 動態範圍

- **色界：** 色彩飽滿、紋樣繁盛。多色並置。華麗、莊嚴
- **空界：** 色彩退潮。紋樣消融。僅餘核心格線的金色微光在深暗底色上
- **真正的空：** 全部消失。純黑。15 秒至 2 分鐘。觀者直面執著

---

## 場域系統

沿用 v3 的二維連續場域：

- **能量 (energy)**: 0-1。活性。影響紋樣密度、線條粗細、色彩飽和
- **秩序 (order)**: 0-1。結構性。影響對稱程度、紋樣規律性、金色線條可見度

距離驅動的變速插值：小幅漂移緩慢（潮水），大幅跳躍前慢後快（質變）。

殘餘條件：下一個場域目標受當前位置影響。70% 鄰近漂移，30% 跳躍。

---

## 生成式紋樣系統

活動帶的核心——用演算法生成唐卡風格的抽象紋樣。

### 紋樣元素

不是粒子。是由弧線段組成的幾何紋樣片段：

- **弧瓣：** 類似蓮花瓣的弧線對，但不具象為蓮花。純幾何的尖頭橢圓弧
- **迴旋線：** 類似火焰紋的捲曲弧線，自我相似的螺旋
- **直射線：** 從中心向外的放射線段，類似唐卡的光背射線
- **環帶：** 同心圓弧段，類似唐卡的裝飾邊帶

### 紋樣行為

紋樣元素由場域驅動：

- **高能量 + 高秩序：** 紋樣繁茂對稱，弧瓣層層嵌套，迴旋線規律排列。曼陀羅般的盛放
- **高能量 + 低秩序：** 迴旋線佔主導，動態劇烈，紋樣快速生成又碎裂。忿怒的能量
- **低能量 + 高秩序：** 只剩少量弧瓣和環帶，極慢極精確地排列。禪定的寧靜
- **低能量 + 低秩序：** 紋樣幾乎消失。偶有微弱的直射線閃現又消亡。寂靜

### 紋樣生命週期

每個紋樣元素有獨立的生命：
- 出生：從無到有漸現（描繪感——像一筆一筆畫出來）
- 存在：持續數秒到數十秒
- 消亡：淡出或碎裂（線段斷開散落）
- 紋樣之間有緣起關係：一個紋樣消亡後，其位置和形態影響下一個紋樣的誕生

## 呼吸節奏

沿用 v3：緣起 → 存在 → 緣滅 → 寂靜 → 再生

### 緣起（Birth）

紋樣從核心向外一圈一圈展開。像唐卡畫師從中心開始往外畫。先出現的是靠近核心的有序紋樣，然後外層逐漸繁盛。

### 存在（Living）

紋樣在場域驅動下持續演化。有些消亡，有些新生。整體紋樣密度和色彩由場域狀態決定。

### 緣滅（Dying）

紋樣從外向內逐圈消融。外層先碎裂消失，中層淡化，最後只剩核心格線。像曼陀羅被掃除。

### 寂靜（Silence）

僅餘核心格線微光。或者——真正的空：全部消失。

### 時間不可預測

每次持續時間由 entropy() × abundance 決定。有的生命 5 秒即滅，有的持續 3 分鐘。

## 核心格線：度量經幾何

基於藏傳唐卡的度量經（造像量度經）的比例系統，取其幾何骨架：

### 結構

- 一個正圓為基礎
- 水平和垂直中線交叉
- 從中心點等分的放射線（8 或 16 等分）
- 同心圓環（按比例遞增：黃金比例或唐卡傳統的身體比例）
- 放射線與同心圓的交點形成網格
- 若干弧線連接特定交點，形成面部、身體的比例暗示——但不成像

### 渲染

- 金色（hsl(45, 85%, 55%)），透明度 0.08-0.15
- 線寬 0.5-1px
- 預計算至離屏 canvas
- 整體有極慢的呼吸：透明度在 0.08 和 0.15 之間以 30-60 秒為週期振盪

## 技術架構

### Canvas 層疊

```
底層：trace canvas（痕跡——前世殘影）
中層：core canvas（核心格線——預計算，極低頻更新）
頂層：main canvas（活動帶紋樣——每幀更新）
```

三層用 CSS z-index 疊加。main canvas 需要透明背景以顯示下方層。

### 極座標系統

活動帶的紋樣用極座標管理：

```javascript
// 每個紋樣元素有：
{
  r: 距離中心的半徑（決定所在環帶）,
  theta: 角度位置,
  type: 'arc' | 'spiral' | 'ray' | 'ring',
  life: 當前壽命,
  maxLife: 最大壽命,
  params: 具體形態參數（弧度、粗細、色彩索引）
}
```

### 效能策略

- 核心格線預計算，不隨幀更新
- 活動帶紋樣數量上限：50-80 個元素（不是粒子——每個元素是由多段弧線組成的紋樣）
- 貝茲曲線用 quadraticCurveTo（2 個控制點），不用 bezierCurveTo（3 個控制點）
- 色彩從預定義調色盤查表，不做即時 HSL 運算
- 遠離中心的紋樣可以降低線條細節

## 情緒基調

莊嚴的敬畏。不是哀愁的無常。

「一切都在變化，而這件事本身就是壯麗的。」

有華麗的段落——然後被抽走。對比越大，教導越深。

---

## 不做的事

- 不畫可辨認的佛教具象符號（佛像、蓮花、金剛杵）
- 不自稱「唐卡」
- 不模仿手繪筆觸質感
- 不用複雜度假裝深度
- 不預設觀者需要佛學知識
- 不讓每一刻都華麗——必須有寂靜和虛空
- 不做成螢幕保護程式——必須有苦的暗流

| 項目 | v3 | v4 |

| 視覺語言 | 粒子（圓點+連線） | 幾何紋樣（弧線+直線+環帶） |

| 色彩 | 場域連續推導 | 預定義礦物色調色盤 |
| 情緒基調 | 寂靜微妙 | 莊嚴敬畏 |

| 保留 | 場域系統、呼吸、痕跡、真空、佛性 | 同（佛性融入核心格線的微呼吸） |
| 移除 | — | 粒子系統、連線、軌跡尾 |
